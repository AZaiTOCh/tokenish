# Engineship

**Engineship** is citizenship-like standing of an **agentic** relative to an **engine** (e.g. tokenish / TOKISH).

Canonical roster: [Agentic Registry](AGENTIC_REGISTRY.md) · Full lexicon: [GLOSSARY](../GLOSSARY.md)

## Values

| Engineship | Meaning |
|------------|---------|
| **Agent** | General / migrated agentic pattern — may have entered as a plug-in or archive-style module; not yet fully specialized as a mother-codebase **Resgent** |
| **Resgent** | Native agentic form — leaner, tighter coupling to this engine; spawned from or programmed for the mother codebase ([RESGENTS](RESGENTS.md)) |

## Why it matters

Like citizenship, Engineship can change: an **Agent** may earn **Resgent** Engineship when finetuned/optimized for a particular engine. See also **[Status](STATUS.md)** (Plug-in · Reserve · Resident) for *how* the agentic sits in the tree today.

## Runtime note (tokenish v0.5.x)

Engineship is **product taxonomy with architectural intent**. Today all registry agentics are in-tree Python modules; a marketplace of foreign Agent plugs is not shipped.
