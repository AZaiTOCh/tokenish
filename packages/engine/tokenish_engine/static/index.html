connection
Auto picks the next working AI for you. Claude or ChatGPT go first if you added their keys. Otherwise Gemini free is preferred, then OpenRouter free models, then Groq, then Perplexity.
model
The specific AI brain. With Gemini we always use gemini-3.5-flash. Groq offers a larger 70b model and a faster 8b model.
API keys
An API key is a private passcode that lets tokenish talk to an AI for you. You only need one. Open Connections at the top to add more with signup links.
status
Green means that AI is ready. Grey means it is unavailable or waiting. Tokenish will auto-switch if one gets busy.
pdf page range
Optional. Example: 12-15 means only send those PDF pages. Leave blank to send the whole PDF (safer for accuracy; may use more tokens).
